## 5. 搭建方式和成本

目前大概是：

```
VSCodium/NeoVim --> 编写文章
Astro/Fuwari --> 博客构建
GitHub && Codeberg --> 版本记录和远端同步
Cloudflare Workers && Codeberg Pages --> 静态部署
```

成本的话，主要就是域名和时间

域名吗，`clina.top` 一年大概 30￥，`150191.xyz` 一年大概 5￥

就这么简单
